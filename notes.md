OBJETO {}
--> É um tipo de dado Javascript. São abstrações ou representações de coisas da vida real 
example: 
const object = {
  name : '';
  phone : ''
  }
--> Cada propriedade do objeto é chamada de atributo

CLASSE
sintaxe: 

class ClassName {
  
}

--> São modelos ou moldes de um objeto 
--> Todo objeto possue comportamentos, ou seja, métodos e funções

MÉTODO 
--> um método JavaScript é uma propriedade de um objeto que contém uma definição de função. 
--> Métodos são funções armazenadas como propriedades do objeto. 


CONSTRUCTOR

--> O método constructor é um tipo especial de método para criar e iniciar um objeto criado pela classe 
--> Só pode existir um constructor por classe
--> Um erro de sintaxe SyntaxError (en-US) será lançado se a classe possui mais do que uma ocorrência do método constructor.
-->Ele é executado assim que a classe é instanciada
